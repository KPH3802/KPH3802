# Kevin Heaney

**Cybersecurity engineer by day, quantitative trading tools by night.**

I work in enterprise firewall operations and build Python-based systems that collect, analyze, and generate signals from public financial data — congressional trades, SEC filings, options flow, economic indicators, and commodity markets.

---

### What I Build

**Market Intelligence Tools** — Automated pipelines that ingest data from SEC EDGAR, FMP, EIA, NOAA, FRED, and Yahoo Finance into local databases, run pattern detection algorithms, and deliver prioritized email alerts.

**Signal Research** — Backtesting frameworks that validate trading signals against years of historical data before anything goes live. The congressional tracker alone covers 46,000+ signals from 297 politicians across 2,700 tickers dating back to 2012.
